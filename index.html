1.需要抓包的手机要有ROOT权限
2.下载tcpdump 并推到手机内/data/local/tcpdump
3.修改文件权限添加可执行adb shell chmod a+x /data/local/tcpdump
4.执行命令抓包
./tcpdump -i any -p -s 0 -w /sdcard/capture.pcap
